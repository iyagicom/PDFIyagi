# PDFIyagi v1.0.0

Linux PDF Viewer + Anonymization Editor (Qt6 Multi-Engine)

PDFIyagi는 빠른 PDF 뷰어 + 문서 비식별화 편집기이다.

이 프로그램은 사전 빌드된 실행 파일로만 제공되며,
소스 코드는 제공되지 않는다.

---

## ✨ 주요 기능

### 렌더링 엔진
- 멀티 엔진 실시간 전환
  - Qt PDF (기본, 빠름)
  - Poppler (텍스트 추출 정확)
  - MuPDF (고품질 렌더링)

---

### 편집 도구
- 텍스트 편집 (오버레이 방식)
- 마킹 도구 (검정 박스 처리)
- 블러 도구 (픽셀화 비식별 처리)
- 이미지 삽입 (클립보드 / 드래그앤드롭)
- 빈 페이지 삽입

---

### 저장 / 내보내기
- 이미지 기반 PDF로 저장만 지원
  - 각 페이지 이미지 렌더링
  - 모든 편집 내용 합성
  - QPdfWriter로 출력

⚠️ 원본 PDF 텍스트 레이어는 유지되지 않음

---

### 텍스트 추출
- Poppler 기반 추출
- 페이지 / 전체 문서 추출 지원

---

## 🧰 도구 모드

| 모드 | 동작 |
|------|------|
| 선택 | 이동 / 크기조절 / 삭제 |
| 텍스트 편집 | 클릭 → 편집 팝업 |
| 마킹 | 드래그 → 검정 박스 |
| 블러 | 드래그 → 픽셀 블러 |

---

## ⌨ 단축키

| 키 | 기능 |
|-----|------|
| Ctrl+O | PDF 열기 |
| Ctrl+S | PDF 저장 |
| Ctrl+Shift+S | 옵션 저장 |
| Ctrl+B | 빈 페이지 삽입 |
| Ctrl+V | 이미지 붙여넣기 |
| Ctrl+Z / Ctrl+Y | 실행취소 / 다시실행 |
| Del | 선택 삭제 |
| Ctrl+T | 현재 페이지 텍스트 추출 |
| Ctrl+Shift+T | 전체 텍스트 추출 |
| Ctrl+Shift+F | 폰트 선택 |
| Ctrl+Shift+C | 마킹 색상 선택 |
| ← / → | 페이지 이동 |
| Ctrl+휠 | 확대 / 축소 |

## 📦 배포 방식

### 릴리즈 형태
- 사전 빌드 실행 파일만 제공
- 소스 코드 제공 없음

---

### Linux 버전
- 포터블 실행 파일 또는 패키지 형태
- 개인 / 상업 / 교육 / 기관 모두 사용 가능
- 재배포 및 패키징 허용

---

### Windows 버전
- Microsoft Store 배포
- 라이선스는 Microsoft Store 정책 적용
