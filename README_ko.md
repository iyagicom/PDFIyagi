# PDFIyagi v0.9.0

**이미지 기반 PDF 처리기 + 빠른 뷰어 (Qt6 멀티 엔진)**

PDFIyagi는 PDF를 **이미지로 재구성하는 방식의 비식별화 전용 편집기 + 빠른 뷰어**다.
모든 저장 결과는 **완전한 이미지 PDF**로 생성되며, 원본 텍스트와 구조는 유지되지 않는다.

---

## 🔥 핵심 특징

* 모든 저장은 **이미지 PDF로 강제 변환**
* 원본 텍스트 / 메타데이터 제거 → **완전 비식별화**
* 화면 그대로 저장되는 WYSIWYG 방식
* 렌더링 엔진 전환 (속도 / 품질 선택 가능)

👉 한 줄 요약
**PDF를 편집하는 것이 아니라, 이미지로 다시 만드는 도구**

---

## ✨ 주요 기능

### 렌더링 엔진

* Qt PDF (빠름)
* Poppler (텍스트 추출 정확)
* MuPDF (고품질 렌더링)

---

### 편집 도구

* 텍스트 오버레이 편집
* 마킹 (검정 박스)
* 블러 (픽셀 비식별 처리)
* 이미지 삽입 (클립보드 / 드래그앤드롭)
* 빈 페이지 삽입

---

### 저장 / 내보내기

* 이미지 기반 PDF만 지원

  * 페이지 이미지 렌더링
  * 편집 내용 합성
  * QPdfWriter 출력

⚠️ 텍스트 선택 / 검색 / 복사 불가
⚠️ 파일 용량 증가 가능

---

### 텍스트 추출

* Poppler 기반
* 페이지 / 전체 추출

## ⌨ 단축키

| 키               | 기능            |
| --------------- | ------------- |
| Ctrl+O          | PDF 열기        |
| Ctrl+S          | PDF 저장        |
| Ctrl+Shift+S    | 옵션 저장         |
| Ctrl+B          | 빈 페이지 삽입      |
| Ctrl+V          | 이미지 붙여넣기      |
| Ctrl+Z / Ctrl+Y | 실행취소 / 다시실행   |
| Del             | 선택 삭제         |
| Ctrl+T          | 현재 페이지 텍스트 추출 |
| Ctrl+Shift+T    | 전체 텍스트 추출     |
| Ctrl+Shift+F    | 폰트 선택         |
| Ctrl+Shift+C    | 마킹 색상 선택      |
| ← / →           | 페이지 이동        |
| Ctrl+휠          | 확대 / 축소       |
