# PDFIyagi v1.0.0

**PDF 비식별화 + 줄 단위 편집 + 이미지 PDF 생성 도구 (Qt6 Multi-Engine)**

PDFIyagi는 문서 내 개인정보를 **완전히 제거**하고,
PDF를 **줄 단위로 직접 편집**할 수 있으며,
결과를 **복구 불가능한 이미지 PDF로 저장**하는 도구다.

---

## 🔥 핵심 사용 구조

PDFIyagi는 **뷰 / 편집 2가지 모드**로 동작한다.

---

### 👁 뷰 모드 (Main Workspace)

대부분 작업은 **뷰 모드에서 수행**된다.

* 초고속 PDF 보기
* 페이지 순서 변경 (드래그)
* 빈 페이지 삽입
* 이미지 붙여넣기
* 블러 (픽셀 처리)
* 마킹 (검정 박스)
* 바로 저장 가능

👉 **편집 없이도 문서 구성 및 비식별화 완료 가능**

---

### ✏ 편집 모드 (Text Only)

텍스트 작업은 편집 모드에서만 수행된다.

* 줄 단위 텍스트 편집
* 문장 수정 / 삭제 / 이동
* 드래그로 줄 재배열
* 구조 복원 기반 편집

👉 **텍스트만 정확하게 다루는 전용 모드**

---

## 🔥 핵심 기능 (Core Features)

### 🔒 비식별화 (Anonymization)

* 블러 (픽셀 처리)
* 마킹 (검정 박스)
* 개인정보 완전 제거
* 이미지 PDF 저장 → 복구 불가

---

### ✏ 줄 단위 편집 (핵심 차별화)

* PDF를 **문장/줄 단위로 완전히 복원**
* 줄 단위 텍스트 편집 (수정 / 삭제 / 이동)
* **드래그로 줄 순서 변경**
* 원본과 동일한 공백 / 줄 구조 유지

👉 **PDF를 워드처럼 편집**

👉 **기존 PDF 편집기와 완전히 다른 방식**

---

## 🚀 왜 다른가

기존 PDF 도구:

* 텍스트 박스 단위
* 레이아웃 깨짐
* 줄 개념 없음

PDFIyagi:

* 줄 단위 구조 인식
* 문장 재구성
* 자연스러운 편집

👉 **문서처럼 편집 가능**

---

## ✨ 주요 기능

### 📄 렌더링 엔진

* Qt PDF (빠름)
* Poppler (텍스트 정확)
* MuPDF (고품질)

---

### 🧰 모드

| 모드 | 설명      |
| -- | ------- |
| 뷰  | 보기 전용   |
| 편집 | 줄 단위 편집 |
| 마킹 | 검정 박스   |
| 블러 | 픽셀 처리   |

---

### 📤 텍스트 추출

* Ctrl+T → 현재 페이지 텍스트 추출
* Ctrl+Shift+T → 전체 텍스트 추출

👉 PDF 구조 그대로 텍스트 생성 (줄/공백 유지)

---

### 💾 저장 / 내보내기

옵션 저장 기능:

* JPEG 저장
* PDF 형식 저장
* 이미지 PDF 저장 (기본)

#### 📄 페이지 범위 지정

* 전체: 0
* 선택: 1,2,3
* 범위: 3-5
* 혼합: 1,2,3-5,8

---

## 🔒 이미지 PDF 저장 (핵심)

* 모든 페이지를 이미지로 렌더링
* 편집 내용 완전 반영
* 텍스트 / 메타데이터 제거

👉 **복구 불가능한 PDF 생성**

---

## ⌨ 단축키

| 키               | 기능            |
| --------------- | ------------- |
| Ctrl+O          | PDF 열기        |
| Ctrl+S          | PDF 저장        |
| Ctrl+Shift+S    | 옵션 저장         |
| Ctrl+B          | 빈 페이지 삽입      |
| Ctrl+V          | 이미지 붙여넣기      |
| Ctrl+Z / Ctrl+Y | 실행취소 / 다시실행   |
| Del             | 선택 삭제         |
| Ctrl+T          | 현재 페이지 텍스트 추출 |
| Ctrl+Shift+T    | 전체 텍스트 추출     |
| ← / →           | 페이지 이동        |
| Ctrl+휠          | 확대 / 축소       |
